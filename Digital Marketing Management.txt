Phase 0. Pre-Requisite Setup

Phase 1. Master Data Management
     - User location data store krne ke liye country, state, city liya hai
	taki pata chale konsa user kaha se hamari system use kr raha hai or kaha usage jyada hai.
	like :
	- Country : India, USA    
	- State : Gujarat, Maharashtra
	- City : Surat, Valsad

Phase 2. User Management
      - Users ke basic details for login taki user register ho to
	users ke social media se thoda real data mil sake.
	- User : system use karne wala person
	- Login : account me enter karega
	- Signup : naya account banega
	- Password : user password
	- Logout : account se bahar nikalne ke liye

Phase 3. Social Media Management
   Social media posts and content manage krne ke liye,
	user login ho gaya to konse platforms hai user ke platforms kitne hai
	kesa content post krega ye sab esme aayega

	- Platform : Instagram, YouTube, LinkedIn
	- Use Platform : user konsa platform use karega
	- Content Type : image, video, text
	- Post Type : reel, story, post
	- Platform Post Type : konsa platform konse post karta hai
	- Post Content Type : kis post me kya content hai
